# S-enfile

fast, secure and free file encryptor in C17. 
## versions

![Windows](https://img.shields.io/badge/Windows-0078D6?style=for-the-badge&logo=windows&logoColor=white) CLI & UI;

![Linux](https://img.shields.io/badge/Linux-FCC624?style=for-the-badge&logo=linux&logoColor=black) CLI only;

![macOS](https://img.shields.io/badge/mac%20os-000000?style=for-the-badge&logo=macos&logoColor=F0F0F0) Nope;

## How to use

### Windows

Open CLI or GUI version .exe, select file or path to it and convert!

### Linux 

Open terminal and type senfile en(encode)/de(decode) path_to_file saving_path
